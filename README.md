# 📁 Portafolio de Evidencias  
## Emprendimiento de Negocios de Software

Este repositorio tiene como finalidad servir como **portafolio de evidencias** de la materia **Emprendimiento de Negocios de Software**. En él se recopila el material desarrollado durante el curso con el propósito de **mostrar y comprobar el aprendizaje adquirido en cada uno de los temas abordados**.

El portafolio incluye trabajos, actividades, proyectos y documentos que reflejan las **metas alcanzadas a lo largo del curso**, los cuales fueron elaborados como parte del proceso formativo y serán utilizados para la **exposición en la muestra académica**.

Cada archivo representa la aplicación práctica de los conceptos vistos en clase, evidenciando la comprensión, el análisis y el desarrollo de ideas relacionadas con el emprendimiento y la creación de negocios basados en software.

### Estructura 
---
El portafolio está organizado en **cinco subcarpetas principales**, cada una correspondiente a una de las **cinco unidades** que se abordan durante el curso. Dentro de cada subcarpeta se encuentran las **metas realizadas en la unidad correspondiente**, debidamente identificadas por su **número de meta**, así como una **conclusión general** que integra y reflexiona sobre los aprendizajes obtenidos en dicha unidad.
